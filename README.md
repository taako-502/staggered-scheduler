# staggered-scheduler

時差を考慮したTodoアプリです。
海外でリモートワークをする人向けに作りました。

URL：https://staggered-scheduler.vercel.app/

## Getting Started

First, run the development server:

```bash
yarn
yarn dev
```

Open [http://localhost:3333](http://localhost:3333) with your browser to see the result.

## Deploy

```bash
vercel deploy
```
